IslamicAI Frontend Test

This page verifies that the IslamicAI frontend can be accessed without requiring login.

Step 1: Build the frontend

Run the following command in the islamic directory:

npm run build

Step 2: Serve the frontend

Run the following command in the islamic directory:

npm run preview

Step 3: Access the frontend

Open your browser and navigate to:

http://localhost:4173

Expected Result

You should see the IslamicAI chat interface without being prompted to log in.

Note

The backend server should be running at http://127.0.0.1:8787 for full functionality.
